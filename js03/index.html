1. 변수를 만드는 작업을 '변수의 선언'이라 부른다.
2. 만들어진 변수에 첫 데이터를 지정하는 것을 '변수의 초기화'라한다.
1,2는 동시에도, 따로도 진행 가능하다.

let test;   // 변수 선언
test = 1;   // 변수 초기화

변수 이름 규칙
변수는 '데이터에 붙이는 이름표' 이름은 기본적으로 '마음대로'지을수 있지만 몇가지 제약 사항이 있다.
1. 변수명은 문자,숫자,&,_ 만 포함 가능
2. 첫 글자로 숫자가 올수 없음
3. 이미 다른 뜻을 가진 단어(키워드)는 불가

데이터 유형을 흔히 '자료형'이라 한다.
