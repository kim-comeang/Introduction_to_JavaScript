함수 - 호출될 수 있는 코드 조각

함수 선언식
function 함수명() {
    // 함수의 기능을 표현한 구문
}

함수 표현식
const 함수명 = function() {
    // 함수의 기능을 표현한 구문
}

함수를 만들고, 함수 호출문으로 사용한다.

함수명();   // 함수 호출


함수 이름을 지을 때 규칙
- 함수의 기능을 적절하게 표현할 수 있는 이름
- 명사보다는 동사
- 소문자로 시작하되, 여러 단오가 섞인 경우 카멜 표기법 사용



함수 선언식, 함수 표현식의 차이
- 문법의 차이
- 함수 선언식은 함수 호출문이 먼저 선언되도 된다. (호이스팅이 가능하다.)
